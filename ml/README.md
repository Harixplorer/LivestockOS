# ML

Machine learning models and pipelines for livestock health intelligence.

## Structure

```
ml/
│
├── data/           # Raw and processed datasets
├── notebooks/      # Jupyter notebooks for exploration & experimentation
├── models/         # Trained model weights and checkpoints
├── outputs/        # Predictions, evaluations, and reports
├── src/            # Core ML source code (training, preprocessing, inference)
└── requirements.txt
```

## Setup

```bash
pip install -r requirements.txt
```
